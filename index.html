HTML PARA SUBIR COM DOCKER

Instruções básicas de passo a passo opcional

  • docker build -t nome_da_imagem .

    • "-t": Nomear images | "nome_da_imagem": Você que define | "." Para diretório atual

  • docker run --name nome_container --rm -id -p porta_local:80 nome_da_imagem

    • "nome_container": Você que define | "--rm": Para remover container após parar execução

    "-id": i - De interativo, d - Executar em Background (Não impedindo prompt)

    "-p": Mapeamento de portas | "porta_local": Você define | "80": Porta da nginx | "nome_da_imagem": Você define

  • Ou use de forma mais simples e rápida - docker run -d -p porta_local:80 nome_da_imagem

    • E por fim, acesse: localhost:porta_local
